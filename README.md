# Weather App

A simple weather application built with EJS that allows users to check weather conditions for various locations.

## Features

- Search for the current weather by city or location
- Display temperature, humidity, wind speed, and other weather details
- User-friendly interface with dynamic EJS templating

## Tech Stack

- **EJS** (Embedded JavaScript Templating)
- Node.js (assumed backend)
- Other technologies as required by your implementation

## Installation

```bash
git clone https://github.com/dineshpushadapu/weather-app.git
cd weather-app
npm install
```

## Usage

1. Start the application:
    ```bash
    npm start
    ```
2. Open your browser and navigate to `http://localhost:3000` (or your configured port).
3. Enter a city or location to get real-time weather updates.
